# Predictive Analytics Using Historical Data

## Project Overview

This project focuses on analyzing historical sales data using Power BI. The dashboard provides insights into sales performance, profit trends, customer segments, and regional analysis through interactive visualizations.

## Dataset Used

* Superstore Dataset

## Tools Used

* Power BI Desktop
* Microsoft Excel/CSV Dataset

## Dashboard Features

* Total Sales Analysis
* Total Profit Analysis
* Total Orders Analysis
* Sales Trend Over Time
* Sales by Region
* Sales by Category
* Sales by Segment
* Sales vs Profit Analysis
* Interactive Region Filter
* Interactive Category Filter

## Key Insights

* Historical sales trends help identify business performance over time.
* Different regions contribute differently to total sales.
* Category-wise analysis highlights top-performing product categories.
* Segment-wise analysis helps understand customer behavior.
* Sales and profit relationships provide useful business insights.
